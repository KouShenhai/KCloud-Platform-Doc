<img src="/img/一键生成项目骨架/img.png"/>

```shell
# 进入laokou-cola目录下
cd laokou-cola
# 执行
mvn clean archetype:create-from-project

cd target/generated-sources/archetype/
# 执行成功，项目骨架会安装本地repository
mvn clean install
```

```shell
# 使用【需要新建一个文件夹】
mvn archetype:generate -DarchetypeCatalog=local
```
